How to run this project?

Open this folder in command line tool.
You need multiple CMD to run this game.
===============================================================
Window 1: Server
- run with the command:

python3 painter_server.py 

===============================================================
Window 2-5: Client
- After the server has been set up. You could run the client in another CMDs with the command:

python3 painter_client.py
